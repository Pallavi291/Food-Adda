# Food Adda Restaurant Website

This is a sample restaurant website called "Food Adda" built using HTML, CSS, and ReactJS. It showcases various features including a homepage with a banner, menu page displaying food items, about page providing information about the restaurant, and a contact page for inquiries and reservations.

## Features

- Responsive design for optimal viewing across devices.
- Dynamic menu rendering using React components.
- Contact form for visitor inquiries.
- Social media integration with Instagram, Twitter, GitHub, and YouTube.
- Clean and organized code structure for easy maintenance and scalability.

## Getting Started

1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd food-adda-restaurant-website`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open your browser and visit `http://localhost:3000` to view the website.

## Technologies Used

- HTML
- CSS
- ReactJS
- React Router
- Material-UI (for icons and styling)
